1. Drag-and-Drop Widgets

Description:

Customize your dashboard with drag-and-drop widgets, resizing and repositioning them as needed.

How to Use:

  • Enter Edit mode using the ✏️ button.
  • Click and drag widgets across the grid.
  • Use handles to resize widgets dynamically.

2. Task Management

Description:

Track active and completed DevOps tasks, and sync them with team activities.

How to Use:

  • Navigate to the Tasks tab.
  • Add new tasks with deadlines, priorities, and statuses.

3. System Monitoring

Description:

Monitor real-time metrics like CPU usage, memory allocation, and network traffic.

How to Use:

  • View widgets such as System Status for detailed metrics.
  • Note: Visual KPIs refresh automatically every X seconds.
